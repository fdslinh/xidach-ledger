# Xì Dách Ledger - Codex Instructions

Build this app according to `/docs/product-requirements.md`.

Use `/docs/design/xidach-ledger-ui-reference.png` as the visual reference.

Priorities:
1. Mobile-first UI
2. Clean Vietnamese interface
3. Simple session entry flow
4. Correct profit/loss calculation
5. Prevent saving invalid sessions where total profit/loss is not 0

Tech stack:
- Next.js App Router
- TypeScript
- Tailwind CSS
- Prisma
- SQLite

Do not over-engineer.
Build the MVP first.
